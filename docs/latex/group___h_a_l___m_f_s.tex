Managed Flash Storage Driver. 

This module implements a managed flash storage able to store a finite number of variable-\/size records. Records are retrieved by their index number.~\newline
 The driver is automatically performs\+:
\begin{DoxyItemize}
\item Wear leveling.
\item Auto repair after power loss.
\item Garbage collection in order to remove erased data.
\end{DoxyItemize}
