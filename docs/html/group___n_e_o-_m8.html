Detailed Description

Ublox GNSS module driver.

Detail will be writen soon

Precondition
In order to use the ADC driver the USE_UBLOX_GPS_MODULE option must be enabled in config.h.

Stateflow

The driver implements a state machine internally, not all the driver functionalities can be used in any moment, any transition not explicitly shown in the following diagram has to be considered an error and shall be captured by an assertion (if enabled).

NEO-M8 Operations

The ADC driver is quite complex, an explanation of the terminology and of the operational details follows.
